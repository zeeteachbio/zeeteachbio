# Zeeteach - Biology Notes & Lectures

A responsive educational website for biology notes and lectures.

## Getting Started

### Prerequisites

-   Node.js installed on your machine.

### Installation

1.  Clone the repository or download the source code.
2.  Open a terminal in the project folder.
3.  Install dependencies:
    ```bash
    npm install
    ```
    *(Note: Do not upload the `node_modules` folder to GitHub. This command creates it for you.)*

### Running the Project

To start the local development server:

```bash
npm run dev
```

### Building for Production

To build the project for deployment:

```bash
npm run build
```

The output will be in the `dist` folder.
